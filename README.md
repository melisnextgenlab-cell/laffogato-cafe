# L'Affogato Retro Kafe - Premium Website

## 🏛️ Proje Açıklaması
L'Affogato, retro konseptli ve profesyonel bir kafe website'idir. Eski bir köşk içinde, ahşap detayları ile ve bahçesiyle müşterilerine nostaljik bir deneyim sunar.

## 🎨 Tasarım
- **Renk Paleti:** Kahve tonları (#6F4E37, #A0826D, #D4A574, #F5F1E8)
- **Font:** Georgia (Serif) - Retro ve Klasik
- **Responsive Design:** Tüm cihazlarda mükemmel görünüm
- **Modern UX:** Smooth scrolling, hover effects, animations

## 📁 Dosya Yapısı
```
laffogato-cafe/
├── index.html      (Ana sayfa)
├── style.css       (Tasarım ve stiller)
├── script.js       (İnteraktif özellikler)
└── README.md       (Bu dosya)
```

## 🚀 Hızlı Başlangıç

### Yerel Olarak Çalıştırma (Hiçbir Kurulum Gerekmez)

1. **Dosyaları İndir:**
   - Sağ üst köşde "Code" > "Download ZIP" tuşuna tıklayın
   - ZIP dosyasını çıkarın

2. **Tarayıcıda Aç:**
   - `index.html` dosyasına çift tıklayın
   - veya sağ tık > "Open with" > Tarayıcı seçin

3. **Değişiklikleri Görüntüle:**
   - Herhangi bir dosyayı text editor'de açın (Notepad, VS Code vb.)
   - Değişiklik yapın ve kaydedin
   - Tarayıcıda F5 tuşu ile sayfayı yenileyin
